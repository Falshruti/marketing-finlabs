⚙️ Setup Required — Connect to CRM

The dashboard is running but needs your CRM credentials. Follow these steps:

  1. 1 Go to api-console.zoho.in → Create a Server-based app → Copy Client ID & Secret
  2. 2 Edit .env file → Set ZOHO_CLIENT_ID and ZOHO_CLIENT_SECRET
  3. 3 Run npm run auth in terminal → Authorize in browser → Copy ZOHO_REFRESH_TOKEN to .env
  4. 4 Restart with npm start → Dashboard auto-loads your leads data
